# 信号 #

信号用来通知线程发生了异步事件，又称为软中断信号，线程之间可以互相发送软中断信号。

## 介绍 ##

这个例子会创建两个线程线程1和线程2，每个线程会分别安装信号，然后给这两个线程发送信号，线程收到信号后会执行对应的信号处理函数。
